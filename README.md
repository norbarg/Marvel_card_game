# How to launch the application

_All commands must be written from the root directory_

##  To create database
`mysql -u root < db/schema.sql`

## To load data
`mysql -u norbarg -p < db/data.sql`

 Password is _securepass_

 ## To install node_modules and start app
 `npm install`
 
 `npm start`
